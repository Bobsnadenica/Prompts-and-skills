1. THE APP ARCHITECTURE

    Target: Kindergarten (ages 3–6). Large buttons, no complex text.

    Tech: Flutter (Dart), sqflite (Local DB), flutter_tts (Auto-read questions), just_audio (Answer sounds), riverpod (State), dio (DLC downloads).

    The Arcade Entry: 3-letter name selection using a retro-style ListWheelScrollView.

    The Game Loop: * Host (Lottie) introduces the question.

        TTS automatically reads the question.

        4 Answers: If an answer is a sound (e.g., "Cow"), it plays the "Moo" when tapped/revealed.

        Correct Answer = Visual reward (Confetti/Stars).

    DLC System: The app reads a manifest.json from a URL. It downloads .zip packs to local storage and integrates them into the game dynamically.

2. THE DLC WEBSITE
